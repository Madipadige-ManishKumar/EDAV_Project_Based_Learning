## 📌 Project Overview
This project explains two important data visualization techniques:
- **Density Plot (Kernel Density Plot)**
- **Contour Plot**

The goal is to help beginners understand how data is distributed and how relationships between variables can be visualized.

  - Sepal length

# 📊 1. Density Plot

## 🔹 What is a Density Plot?
A Density Plot is a **smooth curve** that shows how data values are distributed.  
It is similar to a histogram but **without bars**.

---

## 🔹 Why we use it
- To see **where most data values are concentrated**  
- To understand the **shape of the data**  
- To compare datasets easily  

---

## 🔹 Example
We plotted **sepal length** from the dataset.

### 🧠 Interpretation:
- Peak around **5.5 – 6 cm** → most common values  
- Lower ends → rare values  

---

## 🔹 Advantages
- Smooth and easy to understand  
- Helps identify patterns  
- Good for comparison  

---

## 🔹 Disadvantages
- Does not show exact counts  
- Can be confusing for beginners  

---

# 🗺️ 2. Contour Plot

## 🔹 What is a Contour Plot?
A Contour Plot shows **3D data on a 2D graph** using lines or colors.  
Each line connects points with the **same value**.

---

## 🔹 Simple Idea
Think of a map:
- Each line = same height  
- Close lines = steep area  
- Far lines = flat area  

---

## 🔹 Where it is used
- Weather maps  
- Geography  
- Machine learning  
- Engineering  

---

## 🔹 Advantages
- Shows complex data simply  
- Helps find patterns  
- Good for continuous data  

---

## 🔹 Disadvantages
- Hard to read sometimes  
- Too many lines can confuse  
- Exact values not clear  

---
